Чтобы «перехватывать» ошибки AnyLogic и записывать их в свой файл, вам необходимо создать функцию «onError( Throwable t )» в дополнительном коде класса эксперимента:

 

public void onError( Throwable t ){

       // Здесь можно использовать следующие методы:

       // t.getMessage()

       // t.getStackTrace()

       // ...

}
