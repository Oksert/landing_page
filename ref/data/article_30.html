Система состоит из набора графических инструментов, которые позволяют охватить все аспекты автоматизации бизнес-процесса. В дополнении каждом инструменте предусмотрена возможность описать сложные алгоритмы, описав их на  Java / Javascript.

Статьи данного раздела рассказывают о шагах, которые нужно выполнить, чтобы автоматизировать в системе бизнес - процесс:

  • Добавление и редактирование объектов системы
  • Добавление и редактирование жизненных циклов объектов (workflow)
  • Настройка ролевой модели и меню пользователя
  • Загрузка данных из внешних источников
  • Добавление и редактирование отчетов и дашбордов
  • Настройка печатных форм
  • Описание API  системы
  • Настройка email оповещений
  • Настройка фоновых процессов и алертов (предупреждений по наступлению событий)
  • Описание переноса бизнес-логики с одного стенда на другой
  • Описание внешнего REST-API системы

В каждом пункте есть подробное описание инструмента настройки.
