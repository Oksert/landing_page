В системе можно создавать Таблицы типа "Запрос". Такие таблицы используются для отображения данных, полученных SQL запросом. Чаще всего данные таблицы нужно для формирования аналэтических отчетов. Для создания таблиц типа Запрос необходимо выполнить шаги, описанные ниже.

Вход в систему

  1. В браузере перейдите по ссылке: http://:/#/login
    где   - имя или ip сервера, на котором установлен web application server системы
              - порт, на котором установлен web application server системы

    Например, http://runidea.online/#/login
  2. Войдите в систему под пользователем с ролью Администратор. По умолчанию после установки системы пользователь с правами администратора имеет логин/пароль: admin/admin

                                                                                                                              

Создание таблицы

  1. В верхней части экрана появится Меню, перейдите в пункт меню "Настройка" - "Описание таблиц"
  2. Нажмите кнопку Добавить запись

                                    

    3. В открывшемся окне заполните обязательные поля:

  • Название (название объекта для пользователей системы).
  • Системное имя (внутреннее название объекта в системе, не будет видно пользователям).
  • Тип источника данных (Таблица - для создания объекта, Запрос - для формирования отчетов). В данном случае должен быть указан Запрос
  • SQL Query - запрос для получения данных

                              

    4. При необходимости также можно указать следующие поля:

  • Active condition - в данном поле можно написать условие активности объекта (снимается или ставится галка "Активно" объекта). Данное поле представляет собой выражение на языке Java, преобразующееся к булевой переменной. Если возвращаемое значение условия true, то объект активен, если false - неактивен.

    • Пример условия: 

!"completed".equals(record.getAsString("WORKFLOWSTEPNAME"))
    5. Нажмите кнопку Создать

Активация таблицы

Для того, чтобы результаты SQL запроса отображались на форме необходимо для каждой колонки результатов запроса создать поле таблицы. Для этого:

  • Откройте сецкию поля
  • Для каждого колонки результатов SQL запроса следуйте алгоритму Добавление и редактирование атрибутов объектов

После добавления всех полей установите галку Активно, нажмите кнопку Сохранить

После выполнения данных пунктов можно переходить к Настройке отображения
