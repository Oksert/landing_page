Для каждого объекта Системы, за исключением тех, у которых Тип источника данных Запрос, при установке галки Активно автоматически генерируются два типа форм отображения данных:

  • DETAILS (форма просмотра одной записи)
  • LIST (форма просмотра списка записей)

Для  того чтобы, найти все созданные формы отображения данных объекта Перейдите к пункту меню Настройка->Описание таблиц, кликните по нужно объекту и в открывшейся форме перейдите в секцию Настройки отображения:


В данной статье мы рассмотрим шаги по редактированию формы списка, формы одной записи по умолчанию. Создание и редактирование дополнительных форм отображения данных (таблицы, диаграммы, календари и т.д.) рассмотрено в статье Создание и редактирование отчетов. 

Редактирования формы списка

По умолчанию Системное имя настроек отображения списка (формы списка) совпадает с системные именем объекта.

Редактирование формы списка для всех пользователей доступно только пользователю с правами администратора. При этом каждый пользователь может настроить форму поиска для себя, подробнее смотрите статью Пользовательские настройки отображения списка

Для редактирования формы списка необходимо перейти на форму отображения списка по ссылке: 
http://:/#/<Системное имя display>

где   - имя или ip сервера, на котором установлен web application server системы
          - порт, на котором установлен web application server системы
         <Системное имя display> - системное имя display

Например, для примера выше на портале runide.online ссылка будет иметь вид: http://runidea.onine/#/sm_request

В открывшейся форме списка вы можете настроить:

  • Состав столбцов. Его можно изменять с помощью выпадающего меню, находящегося в правом верхнем углу таблицы
  • Расположение и ширину колонок. их ширину можно изменять с помощью drag'n'drop
  • Группировку данных. Для этого просто перетащите колонку, по которой хотите сгруппировать данные в левый верхний угол таблицы. Группировка может быть множественной, порядок полей группировки можно также менять с помощью drag'n'drop
  • Сортировка данных. Для этого кликните по столбцу, по которому нужно сортировать, для изменения направления сортировки кликните по столбцу еще раз. Сортировка может быть множественной, при этом важно следить за порядком полей, по которым выполняется сортировка.


Для того, чтобы сохранить сделанные  изменения нажмите на кнопку Сохранить форму

Также у формы списка можно настроить возможности по созданию, редактированию и удалению данных непосредственно из списка.
Для этого нужно кликнуть на кнопку "Ключ" в правом верхнем углу таблицы, который приведет к открытию объекта Настройки отображения, где можно выполнить данные настройки, подробнее смотрите статью Настройка отображений в виде списка

Редактирования формы одной записи 

Редактирование формы одной записи доступно только пользователю с правами администратора.

Для того, чтобы отредактировать форму откройте форму одной записи и  кликните на кнопку "Ключ" (видима только пользователям с правами администратора) в правом верхнем углу формы.                                                                                                                                                                                                                                                                                                                                                        

После этого откроется редактор форм, где можно посредством drag&drop можно выполнить следующие настройки:

  • изменения расположения полей, их размера
  • редактирования атрибутов полей (по двойному нажатию на названия поля)
  • добавления, удаление, изменения имени секций
  • изменения порядка секций
  • предпросмотра измененной формы

Редактирование формы поиска

У настроек отображения списка может быть выбран один из двух типов фильтрации: Фильтр или Форма поиска.

Если указан тип фильтрации Форма поиска, то на данной форме появляется форма поиска, где отображаются созданные Поля поиска. Подробнее о полях поиска читайте в статье Настройка полей поиска

Для редактирования формы поиска откройте список и кликните на кнопку "Ключ"  в правом верхнем углу таблицы (доступна только пользователям с правами администратора), который приведет к открытию объекта Настройки отображения.

В открывшейся форме кликните по ссылке Редактировать форму поиска

В открывшемся редакторе форм посредством drag&drap можно выполнить следующие действия:

  • изменения расположения полей, их размера
  • редактирования атрибутов полей (по двойному нажатию на названия поля)
  • предпросмотра измененной формы
