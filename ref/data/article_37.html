Для каждого объекта Системы, за исключением тех, у которых Тип источника данных==Запрос, автоматически при создании генерируется два типа отображение:

  • DETAILS (форма просмотра одной записи).
  • LIST (форма просмотра списка записей).

После того, как у объекта установлен признак «Активно» все типы форм создаются автоматически, на основании свойств атрибутов объектов.

Способ редактирования формы зависит от её типа.

Для редактирования формы просмотра одной записи и формы списка выполните следующие действия:

  1. Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана. Выберите пункт «Описание таблиц».

Редактирование формы поиска

  • У Отображения может быть выбран один из двух типов фильтрации: Фильтр или Форма поиска.
  • На форму поиска попадают только те поля, которые указаны в секции Поля поиска. Для добавления поля поиска перейдите во вкладку Поля поиска и нажмите кнопку добавить запись. В открывшемся окне заполните обязательные поля. Для того, чтобы поле отображалось на форме поиска установите галку Видно пользователю.
  • Для редактирования перейдите в секцию Общая информация и нажмите кнопку Редактировать форму поиска. В открывшейся вкладке совершите действия, аналогичные предыдущему пункту

У формы списка (LIST) есть возможность inline-редактирования, которая позволяет добавлять, удалять и изменять объекты прямо из текущей формы, не открывая форму одной записи.                                                                                                                                                                                                                                                                                                                                        

Для активации этого функционала необходимо:

  1. Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана. Выберите пункт «Описание таблиц».
  2. Откройте описание объекта, в форму которого необходимо внести изменения.
  3. Перейдите в секцию «Настройки отображения», выберите нужное отображение, в открывшемся окне поставьте галки Добавление, Удаление, Обновление.
  4. Нажмите кнопку Сохранить.

Для того, чтобы добавить объекту форму представление в виде иерархического списка необходимо либо добавить новое отображение, либо обновить существующее:

  1. Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана. Выберите пункт «Описание таблиц».
  2. Откройте описание объекта, в форму которого необходимо внести изменения.
  3. Перейдите в секцию Настройки отображения и либо выберите существующую либо добавьте новую. В открывшемся окне поставьте тип отображения TREE_LIST
  4. В Поле иерархии укажите поля, в котором хранится ссылка на родительский объект.
  5. Нажмите кнопку Сохранить
