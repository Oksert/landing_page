Отчет типа «график» представляет собой данные в графическом виде, доступные для выгрузки в формате PNG и PDF. Доступно использование отчетов в дашбордах и на карточках объектов. Пример отчета типа "График" представлен на Рисунке 1. Для более детального просмотра данных за период на нижней временной оси выделите необходимый вам интервал.

  • Нажмите Cохранить
  • В секции Настройки отображения нажмите Создать запись
  • В открывшемся окне выберите в поле Тип отображения MULTI_PLOT, заполните обязательные поля
  • Заполните поля:
    • Имя серии - SERIES
    • Значение по X - XVALUE
    • Значение по Y - YVALUE
    • Поле с типом графика - GRAPH_TYPE
    • Номер группы - GROUP_NUMBER
    • Поле с номером оси Y - YVALUE
  • Откройте секцию Поля поиска
  • Для каждого из @ФИЛЬТР_1...@ФИЛЬТР_N добавьте поле поиска
