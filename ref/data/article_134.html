Добавление редактирование таблиц для ввода данных

У пользователя с правами администратора/разработчика есть возможность как создавать новые объекты системы, так и менять существующие.

Для создания нового объекта систему нужно выполнить следующие шаги:

  1. Создать новую таблицу или сформировать запрос к уже существующим данным
  2. Добавить в созданную таблицу поля
  3. Создать поля-ссылки на другие объекты системы (например, поле для выбора города из справочника)
  4. Создать виртуальные поля, которые отображают данные других таблиц, связанные с текущим объектов (например, все заказы текущего клиента )
  5. Настройка копирования полей из связанных объектов
  6. Настройка кнопок
  7. Настройка правил по расчету значений и свойств полей
  8. Отредактировать формы отображения данных на отдельной форме и на форме списка
  9. Добавить индексы, чтобы ускорить поиск данных и обеспечить необходимые проверки, например, на уникальность полей
