Данное поле позволяет выводить произвольные HTML конструкции и может быть использовано для вывода различных ссылок. Указанный код записывается в значение по умолчанию поля с форматом "HTML метка"

Отдельным примером следует рассмотреть ссылки на другие отчеты:

Ссылка на отчет в модельном окне:

<strong>Показать диаграмму Гантта <a ng-click="openReport(227)"><span class=" glyphicon glyphicon-tasks"></span></a></strong>

где 227 - это код отчета, а классы glyphicon glyphicon-tasks задают типичную иконку отсюда

Ссылка на отчет в новой вкладке:

<strong>Показать диаграмму Гантта в новом окне<a target="_blank" href="{{openInNewTabUrl('report', 227)}}"><span class=" glyphicon glyphicon-tasks"></span></a></strong>

где 227 - это код отчета, а 'report' - тип ссылки (также можно использовать 'entity')
