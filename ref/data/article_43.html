Для добавления в систему нового workflow выполните следующие действия:

  1. Перейдите к пункту меню «Настройки - Жизненные циклы». Нажмите кнопку  Добавить запись.
  2. В открывшемся окне заполните поля (обязательные подсвечиваются красным цветом):
    • Имя (название жизненного цикла).
    • Шаг по умолчанию - Шаг ЖЦ, который будет первым для объекта

                      

      3. Нажмите кнопку Создать.
