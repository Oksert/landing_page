Виртуальные поля – это атрибуты объекта, ссылающиеся на данные других объектов.

Создание формата поля для виртуального поля

Для того,  чтобы создать виртуальное поле в объект [OBJNAME2], которое будет содержать записи [OBJNAME] необходимо:

  1. Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана. Выберите пункт «Форматы полей», подробнее см. раздел Добавление и редактирование форматов полей.
  2. Нажмите кнопку Добавить запись
  3. В открывшемся окне заполните следующие поля:
  • Имя (название формата поля).
  • Связанный экран (пункты 1 - 3)
  • Тип шаблона:
    • vjReadOnly (без возможности добавления записей).
    • vjAddOne (с возможностью добавления записей).
  • Тип поля (VIRTUAL).
  • Стиль поля (subgrid).
  • Таблица (укажите объект из пункта 3).
  • Запрос – укажите запрос, по которому для одной записи объекта [OBJNAME2] можно сопоставить некий набор записей объекта [OBJNAME], например:
"RELEASE="+record.getKeyValue()

       где RELEASE – поля объекта [OBJNAME], record.getKeyValue() – данные текущей записи, т.е. объекта [OBJNAME2].                                                                                                                                                                                                                       

4.     Нажмите кнопку Создать.

Добавление виртуального поля в объект

  1. Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана. Выберите пункт «Описание таблиц». 
  2. Найдите запись об объекте, для поля которого требуется добавить виртуальное поле, сделайте двойной клик по этой строке.
  3. В открывшемся окне перейдите ко вкладке «Поля», нажмите кнопку «Добавить запись».
  4. В открывшемся окне заполните следующие поля:                                                                                                                                                                                                                                                                                                                       
 
  • Название (название поля на форме).
  • Системное имя (название поля в системе).
  • Входит в секцию (укажите название секции, на которой необходимо отображать создаваемое поле – это может быть как уже существующая секция, так и новая – в таком случае она будет создана автоматически).
  • Формат поля (формат поля, созданный в пунктах 4-7).
  • Видно пользователю (значение «Активно»).
  • Активно (значение «Активно»).

Подробнее о добавлении полей см. раздел Добавление и редактирование атрибутов таблиц

    5.  Нажмите кнопку Создать поле
