Создаем таблицу для шаблона по следующим правилам именования:

  1. dbdict: + (пример: таблица шаблона для krtask будет называться krtplkrtask, для contacts - krtplcontacts)
  2. ключевое поле в шаблоне должно называться tpl.name
  3. любые поля, являющиеся атрибутами шаблона (т.е. их значения не нужно переносить в запись) должны называться с префикса tpl. (Например: tpl.is.active)
  4. поля, которые нужно переносить в запись должны называться также как и в целевой таблице

После этого нужно прописать линк (можно посмотреть на примере krexample) на поле tpl.name и кнопка ApplyTemplate (также можно посмотреть на примере krexample).
