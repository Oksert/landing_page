Любой отчет системы можно разместить на форме одной записи как обычное виртуальное поле (подробнее о виртуальных полях читайте в статье  Добавление и редактирование виртуальных полей). При этом можно использовать значения атрибутов записи как поля поиска в запросе к данным, таким образом отображая в отчете только данные имеющие отношение к текущей записи.

Добавление отчета на карточку объекта

Для того, чтобы добавить отчет на карточку объекта, необходимо выполнить следующие действия:

  1. Создайте необходимый отчет согласно статье Создание и редактирование отчетов.

    При необходимости можно использовать поля текущей записи в запросе отчета (аналогично полям поиска), для этого нужно поставить перед названием поля символ @.

    Например, на форме одной записи есть поле Сотрудник и мы хотим отобразить отчет с задачами, где он исполнитель. Для этого нужно в отчете, который будет размещаться на форме одной записи, написать в поле Запрос для выборки данных написать выражение:

    assignedto = @resource

  2. Перейдите к пункту  меню Настройки -> Описание таблиц.
  3. На открывшейся странице найдите запись об объекте, в карточках которого должен отображаться отчет. Откройте эту запись двойным кликом по соответствующей строке.
  4. В открывшемся окне перейдите к секции «Поля». Нажмите кнопку + Добавить запись
  5. В открывшемся окне заполните обязательные поля. Нажмите "+" на поле формат поля
  6. Повторите действия, необходимые для создания виртуального поля, подробнее читайте статью  Добавление и редактирование виртуальных полей)
  7. В поле Связанный экран укажите созданный вами отчет
  8. Нажмите кнопку Создать.
