В данной статье приведены два примера по настройке бизнес-приложений на Idea Platform:

  • Простой пример по настройке приложения для заказа деталей в интернет-магазине
  • Сложный пример по настройке приложения для управления проектами с использованием API системы, настройки инструментов интерактивного планирования и аналитических отчетов

Для каждого примера есть видео-инструкция, где демонстрируется каждый шаг настройки приложения.

Мы советуем посмотреть и попробовать выполнить сначала простой пример, а уже потом перейти к более сложному.
