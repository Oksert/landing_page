Индексы в системе используются для решения двух задач:

  • Ускорение поиска по указанным полям (аналогично индексам в базе данных)
  • Проверка совокупности полей на уникальность.

Создание индекса на одно поле для быстрого поиска данных

Для того, чтобы для поля можно было создать индекс поле должно иметь признак В основной таблице - данный атрибут необходимо указать при создании такого поля.

Для добавления индекса для полей, используемых в системе для поиска, выполните следующие действия:

1.     Перейдите к пункту  меню Настройка->Описание таблиц

2.     Найдите запись об объекте, для поля которого требуется добавить индекс, сделайте двойной клик по этой строке.

3.     В открывшемся окне перейдите к секции «Поля», двойным кликом откройте соответствующую запись о поле.

4.     Нажмите кнопку Создать индекс.

Создание индекса на несколько полей

Для добавления индекса для нескольких полей выполните следующие действия:

1.     Перейдите к пункту меню Настройка->Индексы

2.     Нажмите кнопку Добавить запись.

3.     В открывшемся окне заполните поля 

  • Название
  • Таблица (укажите название объекта в системе для реконсиляции записи которого будет использоваться создаваемый индекс)
  • Тип индекса
    Где возможные значения:
    - INDEX - для быстрого поиска по указанным полям
    - UNIQUE - для проверки уникальности совокупности указанных полей 

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          

4.     Нажмите кнопку Создать.

5.     Перейдите к секции «Поля», нажмите кнопку Добавить запись. Заполните поля:                                                                                                                                                                                                                                                                                       

  • Имя поля - поле, которое будет использоваться в качестве индекса/части индекса)
  • Sort Order - внутренний порядок полей в создаваемом индексе.

7.     Нажмите кнопку Создать, затем кнопку Готово.

8.     Если полей в индексе должно быть несколько, то повторите шаги 5-7 для каждого такого поля.  

Создание первичного ключа реконсиляции

У таблицы должен быть указан один и только один индекс с типом UNIQUE  и установленной галочкой Использовать в качестве первичного ключа реконсиляции.

Для того, чтобы его создать:

1.    Перейдите к секции «Общая информация», установите значение поля Использовать в качестве первичного ключа реконсиляции в значение «Активно».

2.  Нажмите кнопку Сохранить.

3.  Установите значение поля Активно в значение «Активно».

4.  Нажмите кнопку Сохранить, затем Готово.​
