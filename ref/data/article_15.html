Электронная библиотека СУТР РПМ

 Здесь пользователи системы смогу размещать всю необходимую информацию по технологии ремонтов и технического обслуживания путевых машин.

Доступ к статьям электронной библиотеки 

Для доступа к электронной библиотеке выберите в меню пункт "Служба поддержки-справка".

 Все статьи размещены слева в древовидно виде.

Поиск статей

Для поиска нужной статьи наберите нужное слово в поле "Поиск" и нажмите на кнопку "Найти". Если вы хотите искать информацию только по части слова, то введите слово поставив в начале и в конце *. Например, если я хочу найти все статьи со словом "мечт" я должен ввести в строку поиска *мечт*. 
Для поиска по тегам выберите нужные теги из списка (поле находится под строкой поиска) и нажмите на кнопку "Поиск". Поиск возможен одновременно как по тегам, так и по произвольному тексту в строке поиска.

Добавление новых статей

Для добавления новой статьи нажмите на кнопку "Добавить запись", в открывшейся форме заполните поля "Тема" и "Текст статьи", при необходимости укажите родительскую статью и теги (будьте внимательны, возможно тег, который вы хотите ввести уже используется, тогда у вас будет возможность выбрать его из списка, он будет без постфикса (новый)).
