Для создания пользователя:

  1. Откройте браузер и выполните переход по ссылке: http://[serverhostname]/#/login.
  2. В открывшейся странице заполните данные для входа: логин и пароль. Для выполнения следующих действий пользователь, данные для входа которого используются для входа, должен обладать правами администратора.
  3. В открывшемся интерфейсе «Система управления» перейдите к пункту «Настройка - Справочники - Пользователи» панели навигации, расположенной в левой части экрана.
  4. Нажмите кнопку «Добавить запись».
  5. В открывшейся форме заполните следующие поля:
    • Логин – имя для входа в систему.
    • Полное имя – имя пользователя.
    • Профиль – профиль в соответствии с исполняемыми функциями системы (настройка профиля пользователя описана в разделе Профиль пользователя).
    • Пароль – пароль для входа в систему.
    • Email – электронная почта пользователя.
    • Телефон (раб), Телефон (моб): рабочий и мобильный номера телефонов пользователя.
  6. Нажмите кнопку «Добавить запись», затем «Готово».


Для редактирования пользователя:

  1. Откройте браузер и выполните переход по ссылке: http://[serverhostname]/#/login.
  2. В открывшейся странице заполните данные для входа: логин и пароль. Для редактирования информации пользователей редактирующий пользователь должен обладать правами администратора.
  3. В открывшемся интерфейсе «Система управления» перейдите к пункту «Настройка - Справочники - Пользователи» панели навигации, расположенной в левой части экрана (см. Рисунок 3 – Меню «Настройка -Справочники-Сотрудники»).
  4. В списке в нижней части страницы найдите строку с записью о пользователе, данные которого нужно изменить, при необходимости введите используйте фильтры, располженные на колонках таблицы. Сделайте двойной клик по этой строке.
  5. В открывшейся форме внесите правки в требуемые поля.
  6. Нажмите кнопку «Сохранить», затем «Готово».
