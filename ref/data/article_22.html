Создание пользователя

  1. Откройте браузер и выполните переход по ссылке: http://[hostname]:[port_number]/#/login
    где  [hostname] - имя или ip сервера, на котором установлен web application server системы
             [port] - порт, на котором установлен web application server системы
  2. В открывшейся странице заполните данные для входа: логин и пароль пользователя с правами администратора (Если вы только установили систему на свой сервер, то логин/пароль пользователя с правами администратора adm@runidea.online/admin)
  3. В открывшемся окне перейдите к пункту меню Настройка -> Справочники -> Пользователи:

  4. Нажмите кнопку Добавить запись.
  5. В открывшейся форме заполните следующие поля:
    • Полное имя – ФИО пользователя. Этот атрибут используется по умолчанию во всех полях, ссылающихся на пользователя
    • Роль – роль в соответствии с исполняемыми функциями системы (настройка профиля пользователя описана в статье
      Роли пользователей).
    • Email – электронная почта пользователя. Используется как логин для входа в систему.
    • Пароль – пароль для входа в систему

  6. Нажмите кнопку Создать

Редактирование пользователя

  1. Перейдите к пункту меню Настройка -> Справочники -> Пользователи (смотрите рисунок выше)
  2. В списке в нижней части страницы найдите строку с записью о пользователе, данные которого нужно изменить, при необходимости введите используйте фильтры, расположенные на колонках таблицы:


  3. Сделайте двойной клик по этой строке и в открывшейся форме внесите правки в требуемые поля.
  4. Нажмите кнопку Сохранить.
