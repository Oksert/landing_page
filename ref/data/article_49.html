В системе существует возможность создавать несколько типов отчетов:

  • Таблица (LIST);
  • Таблица для дашборда(SIMPLE_LIST);
  • График(MULTI_PLOT);
  • Пирог(PIE_CHART);
  • Столбчатая диаграмма(BAR_CHART);
  • Расписание(SCHEDULER)

Создание отчета

Для создания отчета действия аналогичны созданию объекта, но при создании необходимо выбрать Тип источника данных - Запрос. Для создания отображения отчета перейдите во вкладку Настройки отображения и нажмите Создать запись

В открывшемся окне Тип отображения указывает на то, какой вид будет иметь отчет.

При создании отчета из уже существующей таблицы достаточно добавить новую запись в секции Настройки отображения с указанием необходимого Типа отображения
