Отчет типа «Пирог»

Отчет типа «Пирог» представляет собой данные в графическом виде круговой диаграммы, доступные для выгрузки в формате PNG и PDF. Доступно использование таких отчетов в дашбордах и на карточках объектов. Пример отчета тип "Пирог" продемонстрирован на Рисунке 1.

                                                                                                                                                                                                                                              

                                                                                                                                                                                  Рисунок 1

Для создания отчета типа «пирог» и столбчатая при отсутствии таблицы:

  1. В БД создайте представление вида (шаг можно пропустить, если все необходимые данные содержатся в одной таблице)
create or replace view ИМЯ_ПРЕДСТАВЛЕНИЯ as
select
FTYPE, FCOUNT
from ИСХОДНАЯ_ТАБЛИЦА
where…

Здесь FTYPE – наименование сектора, FCOUNT – числовое значение по этому сектору.

  1. Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана. Выберите пункт «Настройка отображения».
  2. На открывшейся странице нажмите кнопку «Добавить запись».
  3. В открывшейся форме заполните следующие поля:
    • Источник получения данных - Запрос
    • Запрос для получения данных (введите запрос к ранее созданному представлению приведенного ниже вида)
select FTYPE, FCOUNT
from ИМЯ_ПРЕДСТАВЛЕНИЯ
where (ФИЛ_1= @ФИЛЬТР_1 or @ФИЛЬТР_1 is null)
and (ФИЛ_2= @ФИЛЬТР_2 or @ФИЛЬТР_2 is null)
…
and (ФИЛ_ N =@ФИЛЬТР_ N or @ФИЛЬТР_ N is null)
  • Нажмите Создать
  • Создайте поля таблицы FTYPE, FCOUNT
  • В секции Настройки отображения нажмите Создать
  • В открывшемся окне заполните обязательные поля и укажите Тип отображения PIE_CHART или BAR_CHART
  • В поле Группировать полю укажите ftype
  • В поле Агрегирующая функция укажите count
  • В после Аргумент агрегирующей функции укажите fcount
  • Откройте секцию поля поиска. Для каждого из @ФИЛЬТР_1...@ФИЛЬТР_N создайте поле поиска ФИЛЬТР_1...ФИЛЬТР_N
  • Нажмите Сохранить

Создание отчета типа столбчатая диаграмма и пирог от уже существующей таблицы:

  • Настройка - Описание таблиц
  • Выберите нужную таблицу, кликните по ней два раза
  • Перейдите к секции Настройки отображения, нажмите Создать запись
  • Укажите нужный тип отображения
  • В поле Группировать по полю укажите поле таблицы, по которому будет происходить группировка (сектор)
  • Укажите Аггрегирующую функцию
  • Укажите Аргумент аггрегирующей функции
  • Добавьте по необходимости поля поиска, аналогично предыдущему случаю
