Отчет типа multiplot позволяет на одном отчете разместить сразу несколько видов отчетов:

  • Графики (Line)
  • Закрашенные области (area)
  • Столбчатые диаграммы

Пример запроса для построения отчета:

select 0 as xvalue, 0 as yvalue, 'Аналитик' as series, 'bar' as graph_type, 0 as y_axis, 1 as group_number

union all

select 8 as xvalue, 150 as yvalue, 'Разработчик' as series, 'bar' as graph_type, 0 as y_axis, 2 as group_number

union all

select 0 as xvalue, -0.11945249354466797 as yvalue, 'stream0' as series, 'area' as graph_type, 1 as y_axis, cast(null as integer) as group_number 

union all

select 1 as xvalue, 0.12267872875090689 as yvalue, 'stream0' as series, 'area' as graph_type, 1 as y_axis, cast(null as integer) as group_number

union all

select 25 as xvalue, 0.12719473576674103 as yvalue, 'stream3' as series, 'line' as graph_type, 2 as y_axis, cast(null as integer) as group_number

union all

select 26 as xvalue, 0.18791897107024924 as yvalue, 'stream3' as series, 'line' as graph_type, 2 as y_axis, cast(null as integer) as group_number
union all

select 27 as xvalue, 0.17064870690477982 as yvalue, 'stream3' as series, 'line' as graph_type, 2 as y_axis, cast(null as integer) as group_number

где

  • xvalue- значение по оси X
  • yvalue - значение по оси Y
  • series - определяет цвет линии
  • graph_type - тип графика, должен быть одинаков для всех "точек" с одной  series
  • group_number - имеет смысл для точек типа bar, все точки с одним group_number и xvalue отображаются в один столбик (получается столбик с полосками различного цвета)

y_axis - пока не очень понятно на что влияет, замечено, что если сделать метку соответствующему полю "y_axis_1:::y_axis_2", то появляется вторая ось y

Для того, чтобы ограничить области по оси x и y нужно создать поля поиска и задать им нужные значения по умолчанию:

  • x_min
  • x_max
  • y_min
  • y_max
