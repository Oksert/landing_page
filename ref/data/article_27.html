Редактирование справочной информации в системе производится по единой схеме и принципиально не зависит от того, какой конкретно справочник необходимо изменить или дополнить.

Рассмотрим процесс редактирования на примере справочника «Местонахождение».

Для поиска «Местонахождение» выполните следующие действия:

  1. Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана.
  2. Выберите пункт «Справочники - Местонахождение».
  3. На открывшейся странице c помощью значка фильтра на колонках найдите нужную вам запись     Для того, чтобы просмотреть найденные записи, сделайте двойной клик по соответствующей строке в списке записей, удовлетворяющих введенным условиям поиска.

Для создания «Местонахождение» выполните следующие действия: 

  1. Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана. Выберите пункт «Справочники - Местонахождение». 
  2. В открывшейся странице нажмите кнопку «Добавить запись».  В открывшейся форме укажите обязательные поля, они выделены красным цветом

  2.  При наличии дополнительной информации заполните оставшиеся незаполненными поля.
  3.  После внесения необходимых изменений нажмите на кнопку «Создать».

Для редактирования «Местонахождение» выполните следующие действия:

  1.  Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана. Выберите пункт «Справочники - Местонахождение».
  2.  В списке в нижней части страницы найдите строку с записью о необходимом Местонахождении. При необходимости используйте поиск. Сделайте двойной клик по этой строке.
  3.  В открывшейся форме просмотрите и, в случае необходимости, отредактируйте информацию.
  4.  После внесения необходимых изменений нажмите на кнопку «Сохранить».
