Правила (любая бизнес-логика с возможностью изменения текущего и других объектов системы) выполняются до и после перехода между шагами workflow.

Соответственно, если правила до перехода, то объект находится еще в начальном состоянии, а если правило выполняется после перехода, то объект находится в конечном состоянии.

Подробнее о порядке выполнения правил читайте в статье Порядок выполнения правил. 

Добавление правил по переходу между шагами workflow

Для добавления нового правила по переходу между шагами workflow выполните следующие действия:

 Откройте карточку перехода, связь с которым необходимо установить для этих правил.

Перейдите в секцию «Правила», нажмите кнопку +Добавить запись , либо используйте inline редактирование, для этого нажмите кнопку Добавить

                                                                                                                                                                                                                                                                                                                              

В открывшемся окне заполните обязательные поля:  

                                                                                                                                                                                                                                                                                                                                                                                                                                  

  • Правило - правило, которое будет выполняться при переходе workflow
  • Порядок выполнения - порядок выполнения правил по этому переходу
  • Выполнить после перехода - при необходимости выполнять указанное правило после перехода установите в значение «Активно».

Если правило, которое нужно привязать к переходу еще не написано, то кликните на символ +   справа от поля Правило и напишите серверное правило (подробнее о добавлении правил смотрите в статье Серверные правила). После создания правила нажмите на кнопку Назад и вы вернетесь на форму добавления правила и перехода, созданное правило будет подставлено в поле Правило.

Для завершения создания связи перехода и правила нажмите на кнопку Создать, затем Готово
