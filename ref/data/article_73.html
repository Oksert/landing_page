Нам потребуются:

  • Eclipse IDE, начиная с 4-ой версии, я использовал EclipseEE, 4.3.1  http://www.eclipse.org/downloads/
  • JBoss последний версии, я использовал JBoss AS 7.1.1.Final http://www.jboss.org/jbossas/downloads/
Распаковываем архивы куда-нибудь (можно просто в \Users\username\Documents\)

Настройка Eclipse

  • Открываем Eclipse 
  • Создаем в Eclipse новый сервер Ctrl+N и выбираем сервер
  • Ищем в списке сервер JBoss AS 7.1, если не находим, жмем "Download additional server adapters"

1. Находим расширение с названием JBoss AS и начинаем установку
2. Соглашаемся со всеми соглашениями, после установки, если попросят, рестартуем эклипс.
3. После установки расширения выбираем JBoss AS 7.1
4. Если сервер создается первый раз нас попросят создать Runtime. Нам потребуется указать исполнительные файлы сервера в строке "Home Directory"

    5. Оставляем другие параметры без изменений и добавляем наш проект к серверу.

6. Завершаем настройку
7. Добавляем библиотеки сервера в проект. Для этого ПКМ по проекту -> Built Path -> Configure Built Path -> Libraries -> Add Library -> Server Runtime -> Название сервера

8. Проверяем что исчезли все ошибки.
9. Пробуем запустить проект на сервере (Alt+Shift+X, R)
