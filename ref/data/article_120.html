Расположение файла конфигурации

Приоритетное расположение:

<WILDFLY_HOME>/standalone/configuration/system.properties

Расположение по умолчанию:

Внутри WAR файла или папки по адресу:

<WILDFLY_HOME>/standalone/deployments/<deploymentFolder>/WEB-INF/classes/ system.properties

Параметры файла конфигурации

Параметр Описание Значение по умолчанию 
dbConnectionURL Адрес базы данных в формате JDBC
Пример:
dbConnectionURL =jdbc:postgresql://localhost:5432/ip_dev 		  
dbSystemSchema Имя системной схемы (обычно равнозначению system). Для всех СУБД кроме psgsql имя схемы обычно совпадает с именем БД
Пример:
dbSystemSchema=system 		  
dbUserName Имя пользователя для подключения к СУБД   
dbPassword Пароль пользователя для подключения к СУБД   
ipDataPath Адрес основной папки с данными IP в данной папке хранится следующие данные:
  • аttachments – вложения
  • indexes/text – текстовые индексы
  • classlibrary – скомпилированные библиотеки классов
  • SVN/svn-ip.bl – данные и настройки SVN
  • templates – шаблоны выгрузок
 
Для SAAS версии структура папок выглядит аналогично, за исключением того, что эти папки находятся внутри папок рабочих простнанств, т.е.
< ipDataPath >/<wsFolderName>/attachments
< ipDataPath >/<wsFolderName>/indexes/text
… 		Для Windows:
C:/IPData/
Для *nix:
/usr/IPData/ 
javaHomeJDK Расположение Java JDK (используется для компиляции классов ClassLibrary)
Пример:
javaHomeJDK=C:/Program Files/Java/jdk1.8.0_102 		  
javaHomeJRE Расположение Java JRE (используется для выполнения кода на Java)
Пример:
javaHomeJDK=C:/Program Files/Java/jre1.8.0_102 		  
deploymentFolder Имя папки или WAR файла приложения
Пример:
deploymentFolder=deployments/IdeaPlatform.war		  
attachmentMaxSize Максимальный размер вложения в Мб
attachmentMaxSize=25 		10 
isTraceHttpRequestTimeOnФлаг трассировки и времени выполнения http запросов (результаты записываются в лог с маркером HTTP REQUEST).
Пример:
isTraceHttpRequestTimeOn=true 		false 
isStartupTextReindexOn Флаг автоматической переиндексации тестовых индексов при запуске системы.
Пример:
isStartupTextReindexOn =true 		false   
isIpAddressCheckEnabledФлаг привязки токена авторизации к IP-адресу пользователяfalse
