Для того, чтобы включить/отключить язык для входа в систему, нужно сделать следующее:
1. Найти таблицу language (db -> language)
2. Найти интересующий язык.
3. Поставить/снять флаг "Active for Logins".

* Примечание: Для того, чтобы активировать язык, для которого нет форм, сообщений, итп, нужно сначала установить языковой патч.
