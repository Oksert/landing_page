1.     Подготовка дистрибутива

  1. Зайти по RDP на сервер isa.defcont.com:33813 (это IP_SAAS Win Server)
  2. Перейти в папку C:\WebServers\WildFly\wildfly-8.2.1.Final\standalone\deployments
  3. Заархивировать ее содержимое (файл IdeaPlatform_SAAS.war.deployed и папку IdeaPlatform_SAAS.war)

2.     Установка дистрибутива

  1. Зайти по RDP на нужный сервер
  2. Остановить службу "IdeaPlatform CUTR"
  3. Забекапить файл \standalone\deployments\IdeaPlatform_SAAS.war\WEB-INF\classes\system.properties
  4. Удалить содержимое папки \standalone\deployments
  5. Разархивировать дистрибутив
  6. Подменить файл system.properties на забекапленный файл в п.2.1
  7. Запустить службу "IdeaPlatform CUTR"
