В Системе существует механизм автоматического запуска выполнения правил/генерации предупреждений для выполнения в качестве фонового процесса.

Настройки фонового выполнения правила:

  1. Перейдите к пункту «Система управления» панели навигации, расположенной в левой части экрана. Выберите пункт «Настройка – Правила - Задача для Фонового процесса».
  2. Нажмите кнопку +Добавить запись.
  3. В открывшемся окне заполните следующие поля: 
    • Объект настройки («Правило»).
    • Код объекта настройки (внутренний номер правила в системе, которое необходимо выполнить).
    • Время следующего срабатывания (укажите время первого времени запуска правила).
    • Имя фонового процесса (укажите название создаваемого фонового процесса).
    • Повторяемая Работа (укажите нужно ли выполнять это правило регулярно или требуется запуск один раз).
    • Статус
    • Entitytype Id - номер таблицы в системе
    • Entity Id - номер записи в системе

В случае, если выполнение будет регулярным, заполните также следующие поля:

  • Интервал (интервал между повторными выполнениями правила в секундах).
  • Повторять до Даты (если повторять выполнение требуется до определенного момента – укажите этот момент времени).

     4. Нажмите кнопку Создать, затем Готово.
