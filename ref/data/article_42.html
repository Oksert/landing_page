Cистема представляет возможность присвоения объекту Жизненного Цикла,  который позволяет:

  • Создавать схему переходов объекта по шагам ЖЦ
  • Определять права доступа на переходы
  • Определять правила по переходам ЖЦ
  • Создавать оповещения по переходу ЖЦ

После создания и настройки Жизненного Цикла:

  • Перейдите в Настройки - Описание таблиц
  • Выберите запись об объекте, которому необходимо присвоить ЖЦ, кликните два раза
  • В секции Общая информация установите галку Используется workflow
  • Откройте секцию Поля и заполните Значения по умолчанию у полей
    • Work Flow - значение по умолчанию соответсвует id созданного ЖЦ
    • Статус - значения соответсвует id первого шага созданной схемы ЖЦ
    • Название шага - значение по умолчанию соответсвует имени первого шага ЖЦ
