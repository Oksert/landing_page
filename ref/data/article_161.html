Поля поиска позволяют фильтровать данные всех видов отчетов. 

Значения полей поиска могут быть определены следующим образом:

  • Указаны пользователем непосредственно из формы отчета (если у объекта Настройка отображения, соответствующего отчету, установлен Способ фильтрации Форма поиска)
  • По умолчанию. Значением может быть константа или результат выполнения кода на Java
  • Переданы из родительского отчета. Могут быть переданы как поля поиска, так и поля результатов родительского отчета. Подробнее о способах определения поля поиска из родительского отчета читай в статье Настройка переходов в дочерние отчеты 

В данной статье описаны шаги по созданию и определению значений полей поиска.

Создание полей поиска

Для создания поля поиска для отчета перейдите по пункту меню Настройка -> Настройка отображений

Выберите отчет, к которому необходимо добавить поле поиска.

Если необходимо, чтобы поле поиска отображалось и пользователь мог изменить его значение необходимо задать значение поля Способ фильтрации равное Форма поиска.

Далее перейдите на вкладку "Поля поиска" и нажмите на кнопку  +Добавить запись (для создания поля поиска в отдельном окне) или +Добавить  для создания поля поиска из таблицы.

В открывшемся окне заполните поля:

Название - название формы поиска, которое будетотображаться на форме отчета

Системное имя - наименование поля поиска на латинице, которое будет использоваться при настройки бизнес-логики

Формат поля - определяет формат поля и виджет, в котором будет доступно редактирование поля. Если среди существующих форматов поля нет подходящего, то нажмите на кнопку «+» справа от этого поля, и создайте новый формат. Подробнее о настройках форматов читайте в данной статье.

Относится к Экрану - ссылка на объект Настройка отображения, к которому относится данное поле поиска. Если поле поиска создается согласно данной статье, то поле заполнится автоматически

Активно - определяет учитывается ли это поле при отображении формы поиска и настройки бизнес-логики

Видно пользователю - если установлена галка, то поле отображается на форме поиска

Только для чтения - в настоящий момент не используется

Обязательное поле - в настоящий момент не используется

Массив - в настоящий момент не используется

Значение по умолчанию - указывается значение, которым будет заполнено поле при открытии объекта. Необходимо указать хранимое значение, соответствующее типу данных, указанному в поле Формат поля, без кавычек

Вычисление значение по умолчанию - код на языке  Java, в результате которого необходимо заполнить переменную defaulValue, тип переменной должен соответствовать типу данных, указанному в поле Формат поля

После заполнения всех полей нажмите на кнопку Сохранить

Редактирование формы поиска

Для редактирования формы поиска на форме объекта Настройка отображения кликните по ссылке "Редактировать форму поиска":

Далее в открывшемся редакторе форм посредством drag&drop необходимо настроить расположение и размеры полей.

Способы редактирования формы полностью совпадают с редактированием формы одной записи.
