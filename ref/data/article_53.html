Отчет типа «расписание» представляет собой данные в виде календаря с указанием определенных событий в нем. Доступно использование таких отчетов в дашбордах и на карточках объектов.

Для создания отчета типа «расписание»:

  1. Перейдите в пункт в меню Настройки-Описание Таблиц
  2. Нажмите Добавить Запись, заполните обязательные поля
  3. Для каждой колонки результатов запроса добавьте поле таблицы

Пример запроса:

select ou.displayname GROUP2, '' GROUP1, nullif(e.CALENDAREVENT_ID,null) EVENTID
from v_r_orgunit ou
inner join ipcalendar cal on (ou.calendar = cal.calendar_id ) 
left join ipcalendarevent e on (e.calendar_id = cal.calendar_id and e.event_type in ('HOLIDAY','EXCEPTION'))
where ou.parent_orgunit = @DEPT

  • Перейди в секцию Настройки отображения
  • Нажмите Добавить запись
  • В открывшемся окне укажите Тип отображения SCHEDULER
  • При необходимости заполните поле вычисление календаря. Пример:
    • Object calendarId = parentRecord.getAsInteger("calendarid")
  • Если необходимо, добавьте поля поиска, перейдя во вкладку Поля поиска и нажав кнопку Добавить запись
