Установка

Для установки системы необходимо выполнить следующие действия:

  1. Скачать установочный пакет, который содержит в себе 2 файла:
    • дамп БД;
    • архив с дистрибутивом приложения (веб-сервер и веб-приложение).
  2. Получить параметры доступа к инстансу СУБД MySQL.
  3. Настроить базу данных:
    • Разверните дамп базы данных из установочного пакета.

Создайте учетную запись с правами на создание/редактирование/удаление таблиц, данных в них, индексов.

  1. Установить веб-сервер и приложение:
    • Установите Java JRE и JDK (версии 7.XX, например, 7.79).
    • Распакуйте архив с веб сервером в любую папку на диске (Например, «C:\WebServers\», в таком случае конечная структура будет такой: «C:\WebServers\WildFly»).
  2. Настроить подключение к БД:

Настройки подключения к БД задаются в файле \standalone\deployments\IdeaPlatform.war\WEB-INF\classes\hibernate.cfg. В папке classes находятся примеры подключения к различным СУБД. Необходимо задать следующие параметры подключения:

  • hibernate.connection.driver_class - java класс, используемый для подключения к СУБД ;
  • hibernate.connection.url - адрес БД (в случае, если в connection.url указывается порт, то необходимо проверить, что этот порт открыт firewall'ом и прослушивается СУБД);
  • hibernate.connection.username - логин учетной записи для подключения к БД;
  • hibernate.connection.password - пароль учетной записи для подключения к БД;
  • hibernate.default_schema - БД по умолчанию;
  • hibernate.dialect - диалект (нужно просто скопировать из примеров в папке classes или посмотреть на http://www.tutorialspoint.com/hibernate/hibernate_configuration.htm).

Ниже приведен пример настроек подключения к БД на СУБД MySQL:

com.mysql.jdbc.Driver

jdbc:mysql://127.0.0.1:3306/ip02

root

P@$$w0rd9m77

ip02

org.hibernate.dialect.MySQLDialect

    3. Изменить настройки системы в БД:

В таблице ipsysparam для записей со значениями поля «PARAM_NAME» «server.java.home.jre» и «server.java.home.jdk» укажите директории на сервере, в которые были установлены Java JRE и Java JDK соответственно на шаге 2 данной инструкции, например, «C:\Program Files\Java\jre7» и «C:\Program Files\Java\jdk1.7.0_79».

    4. Настроить веб-сервер (WildFly).

Для дополнительной (но не обязательной) настройки веб-сервера перейдите по ссылке http://[serverhostname]:9990/console/App.html#home, на странице логина введите логин/пароль admin/IPAdmin и произведите необходимые изменения в настройках (в том числе измените порт, по которому будет доступен веб-сервер, по умолчанию используется порт 8080), при необходимости воспользуйтесь руководством по настройке, ссылка на которое будет указана на стартовой странице системы по настройке веб-сервера. Для добавления/изменения пароля пользователя admin необходимо из папки /bin выполнить команду add-user.bat далее следовать инструкциям (см. Рисунок 2 – Настройка веб-сервера). При этом пользователь должен состоять в группе ManagementRealm.

Настройка веб-сервера

При первичной настройке необходимо:
  • Задать общий вид адреса системы.
  • Настроить подключение к SMTP-серверу.
  • Указать адрес электронной почты, с которой будет производиться рассылка.
  • Указать настройки подключения через https.
  • Настроить уровни логирования.
  • Указать диапазон IP-адресов, по которым будет доступна система.

    5. Создайте службу в операционной системе.

Из директории «WildFly/bin» через командную строку выполните команду «service.bat install», перезагрузите операционную систему.

Запуск системы

Для запуска системы:

1.     Войдите на сервер под учетной ролью, обладающую права администратора.

2.     Откройте программу «Службы» и запустите службу «Idea Platform».

3.     Откройте в браузере на сервере страницу по адресу http://[serverhostname]:[port], где [serverhostname] - доменное имя сервера, на котором установлена система; [port] - номер порта, указанного при настройке веб-сервера. Таким образом вам будет доступна страница логина.
