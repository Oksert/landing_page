О платформе Idea Platform

Idea Platform - российская платформа для быстрого создания масштабируемых веб-приложений.

Платформа позволяет быстро описать модель данных, настроить интерфейс для редактирования и просмотра данных с учетом ролевой модели, прикрепить один или несколько жизненных циклов к созданным объектам, создать рабочие места для пользователей с необходимым набором оперативных и аналитических отчетов. Таким образом, в платформе может быть описан бизнес-процесс любой сложности.

Для реализации данных функций реализован набор графических инструментов, которые подробно описаны в данной справочной документации.

Доступ к платформе возможен онлайн (нужно создать собственное пространство в нашем облачном сервисе runidea.online) или скачать дистрибутив и установить платформу на собственном сервере (возможна установка на CentOS Linux и Windows, дополнительного платного ПО не требуется).

О справочной документации

В справочной документации подробно раскрыты все вопросы, необходимые для настройки веб-приложений на Idea Platform:

  • Установка системы
  • Обновление системы
  • Администрирование системы
  • Описание инструментов настройки бизнес-логики
  • API для работы с объектами системы
  • REST-API для настройки интеграций с внешними системами
  • Загрузка данных из внешних источников
  • Настройка шаблонов документов и печати, настройка отправки электронной почты
  • Примеры по настройке приложений на Idea Platform
  • Средства импорта/экспорта бизнес-логики

Навигация по справочной документации

Для навигации по справочной документации можно пользоваться:

  • Древовидной панелью в левой части страницы Документация
  • Ссылками на дочерние и связанные статьи

Древовидная панель навигации выглядит следующим образом:

Для того, чтобы перейти на дочернюю статью (у каждой статьи в заголовке приведен список дочерних статье) или связанную по смыслу статью .

Например, кликнув на пункт Установка системы данной статьи вы попадете в статью с описанием шагов по установке Системы.

При необходимости вернуться в предыдущую статью просто воспользуйтесь кнопкой "Назад" браузера.
