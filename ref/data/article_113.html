Для RPM:

ExcelTemplateEngine ete = new ExcelTemplateEngine();
ete.createTemplate(<tableName>);
ete.writeToFile(<out excel file>);

Для SAAS:

import ru.ip.server.office.ExcelTemplateEngine;

ExcelTemplateEngine ete = new ExcelTemplateEngine();
ete.createTemplate(<entityTypeId>);
ete.writeToFile(<out excel file>);

<out excel file> - это полный пусть для файла.

Вот пример кода для SAAS, развернутого на Linux:

ru.ip.server.office.ExcelTemplateEngine ete = new ru.ip.server.office.ExcelTemplateEngine();
ete.createTemplate(987);
ete.writeToFile("/usr/IPData/SAAS/ws60_cwzxydqi/stock_operation.xslx");

Для того, чтобы забрать файл с Linux - сервера на локальную Windows машину нужно:

  1. Установить PUTTY (вместе с пакетом pscp)
  2. Открыть командную строку и перейти в директорию установки PUTTY
  3. Выполнить команду:   pscp <linux_user>@<linux_ip>:<directory_to_file> <Windows_directory>

      Например:

pscp root@94.142.142.227:/usr/IPData/SAAS/ws60_cwzxydqi/stock_operation.xslx C:\1
