Установка

Скачиваем тулы для сборки из исходников

sudo yum groupinstall "Development Tools"

sudo yum install gettext-devel openssl-devel perl-CPAN perl-devel zlib-devel curl-devel

sudo yum install curl-devel

Скачиваем исходники в папку tmp

cd /tmp

wget https://github.com/git/git/archive/v2.13.2.tar.gz

Распоковыывам исходники

tar -zxf v2.13.2.tar.gz

Выполняем сборку и установку

cd git*

make configure

./configure --prefix=/usr/local

sudo make install

Проверить, что все поставилось:

git --version

Настройка подключения к Git

Если импорт через ssh

Выполнить команду 

git clone https://....

зайти в <git repository>/.git

nano config

Прописать путь формате:

 url = https://userName:Password@bitbucket.org/VadimDjan/ip-logic-rpm.git

Если импорт через ssh

Узнать есть ли на хосте публичный RSA ключ

cat ~/.ssh/id_rsa.pub

Если что-то вернулось - вставить это в список достоверных ключей для BitBucket (https://confluence.atlassian.com/bitbucket/add-an-ssh-key-to-an-account-302811853.html )

Если нет - нужно сгенерить новый ключ

ssh-keygen -t rsa
(при геренации не забыть записть кодовое слово)

После этого выполнить шаг с регистрацей ключа в BitBucket

https://confluence.atlassian.com/bitbucket/add-an-ssh-key-to-an-account-302811853.html

Проверить подключение до репозитория

ssh -T hg@bitbucket.org

(при необходимости ввести кодовое слово)

Настройка постоянной аутентификации в Git

Выполнить первичный checkout из репозитория (обычно в папку /usr/IPData/)

git clone git@bitbucket.org:VadimDjan/ip-logic-rpm.git

Ввести пароль и после командой ls проверить, что папка появилась

Отключить постоянное требование кодового слова при выполнении команд git

eval $(ssh-agent)

ssh-add -k ~/.ssh/id_rsa

Настройка параметров системы

В настройках IdeaPlatform задать строку с адресом репозитория

versionControlSystemRoot=/usr/IPData/ip-logic-rpm
