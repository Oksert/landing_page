В системе доступна возможность перехода из одного отчета в другой. Для того, чтобы настроить такие переходы, необходимо:

Перейдите к пункту  меню Настройка->Настройка отображения

На открывшейся странице в списке записей найдите отчет, из которого будет осуществляться переход в дочерние отчеты. Сделайте по этой строке двойной клик.

В новом окне перейдите в секцию «Связи с другими отображениями». Нажмите кнопку + Добавить запись либо используйте inline редактирование:                                                                                                                                                          

В новом окне укажите, если необходимо поле для клика. Это настройка приведет к тому, что дочерний отчет будет открыт только по клику на конкретную ячейку, а не строку в целом. При этом для одного отчета могут быть настроены как по клику на ячейки, так и по клику на строку.

Нажмите кнопку Создать . Далее перейдите в секцию Меппинг полей. Нажмите кнопку Добавить (также возможно создание через inline редактирование).

В открывшемся окне заполните поля:

Поле дочернего отображения - это поле поиска дочернего отчета, которое необходимо заполнить

Способ расчета - способ, которым будет заполнено указанное Поле дочернего отображения.
Возможны три варианта:

  • PARENT_FIELD - это значит, что Поле дочернего отображения  будет заполнено одним из значений полей родительского отчета
  • PARENT_QUERY_FIELD -  это значит, что Поле дочернего отображения  будет заполнено одним из значений полей поиска родительского отчета
  • CONSTANT - это значит, что Поле дочернего отображения  будет заполнено константой   

Поле родительского отчета - поле заполняется если выбран Способ расчета PARENT_FIELD или PARENT_QUERY_FIELD, в данном поле соответственно указывается ли поле отчета, либо поле поиска отчета.

Значение по умолчанию - поле заполняется константой, если выбран Способ расчета CONSTANT.                                                                                                                                                                                                                                                                                                                    После заполнения поле нажмите кнопку Сохранить
