В данном разделе будут описаны средства работы с API системы, которые помогут настроить бизнес-логику:

  • Серверные методы и классы. Данное API позволяет делать запросы, создавать, изменять и удалять объекты системы, а также помогает выполнять сложные расчеты, например, с использованием производственного календаря. 
  • Клиентские методы и классы. По сути дублирует серверное API для использования в клиентских правилах.
  • Пользовательские классы (Class Library). Позволяет создавать собственные классы, таким образом позволяя использовать в настройке бизнес-логики код на JAVA любой сложности, а также многократно переиспользовать однажды написанные методы.
