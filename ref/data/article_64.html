Помимо уже существующих методов, которые можно использовать при настройке системы, существует возможность добавлять и собственные. Для того, чтобы создать собственный класс (объект ClassLibrary), выполните следующие действия:

  1. Перейдите к пункту «Система управления» панели навигации, расположенной в левой части экрана. Выберите пункт «Настройка- Правила – Class Library».
  2. Нажмите кнопку +Добавить запись. В открывшемся окне заполните обязательные поля:
    • Имя Класса (название класса на латинице, в дальнейшем будет использоваться для доступа к методам данного класса).
    • Код (код на языке Java, содержащий в себе реализацию необходимых серверных методов, см. пример ниже).

Пример кода в Class Library:

public static String getRequirementInitiatorFromComment(EntityDTO comment) throws Exception{
        EntityDTO req = QueryUtils.getRecord("SD_REQUIREMENT", "SD_REQUIREMENTID = " + comment.getAsInteger("ENTITYID"));
        if (req.get("INITIATOR")!=null){
               EntityDTO initiatorUser = QueryUtils.getRecord("USER", "USERID = " + req.getAsInteger("INITIATOR"));
               return initiatorUser.getAsString("FULLNAME");
        }
        return "не указан";
}

    3. Нажмите кнопку Создать, затем кнопку Готово.

    4. Дважды перезапустите на сервере службу «Idea Platform».

  • Рекомендуется для отдельных модулей системы создавать отдельные классы. Доступ к методу из Class Library осуществляется по шаблону: «Lib.[Class_Name].[Method_Name]», где [Class_Name] – значение поля «Имя Класса» объекта Class Library, содержащего необходимый метод,
  • [Method_Name] – название метода в коде указанного класса. Например, код для доступа к методу getRequirementInitiatorFromComment(EntityDTO comment)  из класса SDUtils будет выглядеть следующим образом:

Lib.SDUtils.getRequirementInitiatorFromComment(someEntityDTO)

Для того, чтобы изменить существующий класс (объект ClassLibrary), выполните следующие действия:

  1. Перейдите к пункту «Система управления» панели навигации, расположенной в левой части экрана. Выберите пункт «Администрирование - Правила – Class Library».
  2. В списке записей найдите необходимый класс, двойным кликом по строке откройте запись с информацией о нем.
  3. В открывшемся окне при необходимости отредактируйте следующие поля:
    • Имя Класса (название класса на латинице, в дальнейшем будет использоваться для доступа к методам данного класса).
    • Код (код на языке Java, содержащий в себе реализацию необходимых серверных методов, см. пример выше).
  4. Нажмите кнопку Сохранить, затем кнопку Назад.
  5. Дважды перезапустите на сервере службу «Idea Platform».
