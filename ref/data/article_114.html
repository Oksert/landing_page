1. Установить postgresql 9.4

2. Создать пустую базу данных и развернуть дамп, подробнее см. здесь:

http://sutrrpm.ru/#/reference?entityId=348&rowId=101

3. Установить java jdk версии 7_79 (не меняя директории установки по умолчанию)

4. Установить значение переменной системного окружения  JAVA_HOME

5. Установить Wildfly 8.2.1  (Для этого необходимо распокавать архив с Wildfly в директории C:/webservice)

6. Со стенда разработки скопировать файл IdeaPlarform.war (файл лежит в директории job Jenkins, который выполяет автоматическую сборку приложения)

7. Поменять разрешения файла с war на zip

8. Распокавать архив в любую директорию, зайти в папку WEB-INF/classes и открыть файл system.properties

9. Отредактирвать параметр db_connection в файле system.properties установив правильные hostname, port и название базы данных, созданной на шаге 2

10. Заархивировать всю папку (стандартным архиватором Windows) и сменить разрешение файла на war,  в результате должен получиться файл с названием IdeaPlatform_SAAS.war

11. Запустить командную строку, перейти к директории \bin\service и запустить команду:

service.bat install

12. Заменить файл standalone.xml (находится в директории  \standalone\configuration) на аналогичный файл со стенда разработки

13. Запустить сервис Wildfly, созданный в.11 данной инструкции
