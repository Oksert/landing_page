Отчет типа «Простая таблица» представляет собой данные в табличном виде, доступные для выгрузки в Excel. Доступна настройка группировки колонок в заголовке отчета, группировка и сортировка по колонкам, изменение порядка и ширины колонок, использование отчетов в дашбордах и на карточках объектов.

Отчеты могут быть созданы как и из обычной таблицы так и из таблицы Типа запрос. В случае когда необходимо отфильтровать по какому-либо признаку данные хранящиеся в таблице используется вариант создания отчета из существующей таблицы, с указанием при этом условия выборки данных на языке SQL, в иных случаях используется вариант создания таблицы типа Запрос.

Пример отчета Простая таблица:                                                                                                                                                                                                                                                                                                                                                                                                           

Создание отчета типа Простая таблица и Таблица для дашборда из новой таблицы типа Запрос

  1. В БД создайте представление вида (шаг можно пропустить, если все необходимые данные содержатся в одной таблице):
create or replace view ИМЯ_ПРЕДСТАВЛЕНИЯ as

ПОЛЕ_2,

ПОЛЕ_N
from ИСХОДНАЯ_ТАБЛИЦА
  • Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана. Выберите пункт «Описание таблиц».
  • На открывшейся странице нажмите кнопку «Добавить запись».
  • В открывшейся форме заполните обязательные поля. В поле Тип источника данных поставьте Запрос
  • В поле sql query:
select ПОЛЕ_1, ПОЛЕ_2,

where (ФИЛ_1= @ФИЛЬТР_1 or @ФИЛЬТР_1 is null)
and (ФИЛ_2= @ФИЛЬТР_2 or @ФИЛЬТР_2 is null)
…
and (ФИЛ_ N =@ФИЛЬТР_ N or @ФИЛЬТР_ N is null)

  • Для каждого из ПОЛЕ_1 .. ПОЛЕ_N создайте поле таблицы
  • Нажмите Сохранить 

                                                                                                                                                                                                                                                      

  • Перейдите на секцию Настройки отображения и нажмите Создать запись
  • В открывшемся окне заполните обязательные поля
  • Нажмите Сохранить
  • При необходимости установите способ фильтрации - Форма поиска
  • Выберите необходимый тип отображения:
    • LIST простая таблица

    • TREE_LIST иерархический список
  • Для каждого из @ФИЛЬТР_1...@ФИЛЬТР_N создайте поле поиска ФИЛЬТР_1...ФИЛЬТР_N
  • Для того, чтобы проваливаться из отчета в форму одной записи необходимо следовать одну из сценариев
    • Выберите нужный экран в поле Экран просмотра одной записи. Выберите поле таблицы, в котором хранится ключ, по которому однозначно определяется запись в поле Поле со значением ключа.

Раскраска отчета

Для того чтобы строки отчетов типа «простая таблица» или «таблица для дашборда» имели нестандартную окраску измените шаблон Запроса для получения отчета на следующий:

select ПОЛЕ_1, ПОЛЕ_2,
…
ПОЛЕ_N, COLOR
from ИМЯ_ПРЕДСТАВЛЕНИЯ
where (ФИЛ_1= @ФИЛЬТР_1 or @ФИЛЬТР_1 is null)
and (ФИЛ_2= @ФИЛЬТР_2 or @ФИЛЬТР_2 is null)
…
and (ФИЛ_ N =@ФИЛЬТР_ N or @ФИЛЬТР_ N is null)

где COLOR содержит значение цвета в формате шестнадцатеричного кода с использованием маски #RRGGBB, например: «#f0f8ff», «#adff2f» или «#f6546a».

Кроме того, необходимо создать поле результата с именем поля «COLOR» и кодом формата поля «Строка». Соответствующий столбец на форме просмотра результатов необходимо скрыть.

Для окраски ячеек для ПОЛЕ_N необходимо создать колонку ПОЛЕ_N_COLOR, далее действия, аналогичные предыдущему пункту

Создание отчета типа Простая таблица и Таблица для дашборда из существующей таблицы

В случае, когда необходимо отфильтровать данные, изменить порядок или состав колонок уже существующей таблицы можно не создавать новую таблицу, а к уже существующей добавить новую Настройку отображения. Пусть есть таблица Запросы на доработку, которая имеет атрибут инициатор. Для каждого пользователя необходимо создать отчет, который будет содержать только те запросы, которые он инициировал.

  1. Перейдите в Настройки - Описание таблиц
  2. Найдите нужную вам таблицу, кликните по ней 2 раза
  3. В открывшемся окне перейдите ко вкладке Настройки отображение                                                                                                                                                                                                                                                                                      
  4. Нажмите кнопку Добавить запись
  5. В открывшемся окне заполните поля:                                                                                                                                                                                                                                                                                                                    
  • Системное имя - имя отображения в системе
  • Тип отображения
  • Запрос для выборки данных - условие на языке SQL, которому должны удовлетворять все записи, попадающие в отчет
  • Шаблон заголовка - Имя отображения
  • Способ фильтрации - Форма поиска или Фильтр
  • Экран просмотра одной записи - Форма одной записи при клике на строку отчета
  • Поле со значенем ключа - указывает Форма записи какого объекта должна открываться при проваливании
