Архитектура платформы

Система состоит из следующих базовых компонентов:

  • Система управления базами данных. Текущей базовой СУБД является СУБД с открытым исходным кодом PostgreSQL, которая обладает возможностями по развертыванию кластера для обеспечения отказоустойчивой, а также возможностью репликации данных для разделения транзакционных (OLTP) и аналитических (OLAP) запросов
  • Сервер приложений. Сервер приложений представляет собой Java EE приложение, развернутое на WildflyApplication Server (веб-сервер приложений с открытым кодом), который поддерживает развертывание кластера из нескольких Application Server-ов, что также обеспечивает отказоустойчивость работы системы
  • Веб-клиент. Веб-приложение (сайт), запускаемое через обычный браузер. Системой поддерживается Топ 5 популярных браузеров, которые могут быть запущены на всех типах популярных пользовательских операционных систем (Windows, Mac OS, Linux, Android, iOS)

Поддерживаемые системное ПО

Систему можно установить на следующие операционные системы: ОС CentOS Linux (рекомендованная версия CentOS Linux 7 (Core)) и ОС Windows (рекомендованные версии Windows Server 2008R2 или Widows Server 2012R2).

В качестве СУБД используется PostrgreSQL 9.4 и старше (рекомендуемая версия PostrgreSQL 9.6).

В качестве веб-сервера приложений используется Wildfly (не требует отдельной остановки, так как включен в дистрибутив приложения).

В качестве веб-сервера используется  Nginx.
