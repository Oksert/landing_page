После создания новой таблицы в нее по умолчанию копируются поля из таблицы, указанной в качестве шаблонной (поле "Базовой таблица" на форме Таблицы). В данной статье описаны шаги по  созданию новых и редактированию текущих полей таблицы.

Создание нового поля таблицы

Для добавления новые полей выполните следующие действия:

  1. Перейдите к пункту «Настройка» панели навигации, расположенной в левой части экрана. Выберите пункт «Описание таблиц».
  2. Найдите запись об объекте, в который требуется добавить поле, сделайте двойной клик по этой строке.
  3. В открывшемся окне перейдите ко вкладке «Поля», нажмите кнопку Добавить запись                                                                                                                                                                                                                                                                                                                                                                                                                                                
  4. В новой форме заполните поля :                                                                                                                                                                                                                                                                                                                                          

  • Название - название поля на форме
  • Системное имя - внутреннее название поля в системе, не будет видно пользователям
  • Массив - если установлена галка, то поле имеет возможность множественного выбора
  • Формат поля - определяет формат поля и виджет, в котором будет доступно редактирование поля. Если среди существующих форматов поля нет подходящего, то нажмите на кнопку «+» справа от этого поля, и создайте новый формат
  • В основной таблице - определяет в какой из двух таблиц будет создано поле: основная таблица или таблица атрибутов. Если по данному полю подразумевается частый поиск данных, то лучше установить галку у этого поля и, тем самым, определить его в основную таблицу. Далее на данное поле можно будет повесить индекс. Значение по умолчанию "True"
  • Видно пользователю - если установлена галка, то поле показывается на форме просмотра одной записи
  • Показывать в списке записей -  если устанавливается галка, то поле нужно показывается на форме списка
  • Обязательное поле - если установленная галка, то без заполнения этого поля нельзя создать или изменить запись объекта
  • Аудит включен - если установлена галка, то по полю нужно будет вестись история – для этого также требуется установить соответствующий признак в описании объекта системы
  • Только для чтения - если галка установлена, то поле будет недоступно для редактирования
  • Отобразить на форме поиска - если галка установлена, то по этому полю будет производится поиск
  • Формат поля на форме поиска - указывает, какой формат будет иметь данное поле при поиске по нему, по умолчанию проставляется такой же как Формат поля  
  • Значение по умолчанию  -  указывается значение, которым будет заполнено поле при открытии объекта. Необходимо указать хранимое значение, соответсвующее типу данных, указанному в поле Формат поля, без кавычек
  • Входит в секцию - определяет вкладку, на которой будет размещено поле. Необходимо указать название уже существующей секции на форме просмотра объекта или название новой секции. Если это поле не будет заполнено, то по умолчанию новое поле будет добавлено в первую вкладку формы одной записи.
  • Вес поля при поиске по тесту - определяет приоритетность поля при полнотекстовом поиске. По умолчанию устанавливается 1. Имеет смысл только если по таблице данного поля включен полнотекстовый поиск
  • Перенос по словам  - данная галка имеет смысл только для полей типа "Tекст" и определяем каким-образом данные будут отображаться на форме списка. Если  галка установлена, то данные будут отображаться полностью путем переноса слов по словам, расширяю строку таблицы, если не установлена, то будет отображаться только та часть строки, которая помещается в ширину столбца, а в конце будет стоять знак "..."
  • Единица измерения - единицы измерения, в которых указаны значения данного поля

5. После внесения всех данных нажмите на кнопку Создать поле, перейдите в секцию Общая информация, Нажмите Сохранить 

6. После создания поля будут автоматически заполнены поля :

  • Тип поля
  • Номер пула
  • SQL Таблица
  • SQL Имя поля 
  • Внутреннее имя

Просмотр и редактирование полей таблицы

Отредактировать свойства существующих полей может только пользователь с правами администратора/разработчика.

Для того, чтобы просмотреть и отредактировать свойства существующего поля можно воспользоваться одним из 2-х способов:

  • Найти нужное на вкладке поля объекта Таблица (смотрите шаги 1-2 выше) и дважды кликнуть по полю
  • Открыть форму одной записи объекта, кликнуть по иконке "Шестеренка"  , в открывшейся форме дважды кликнуть по нужному полю                                                                                                                                                                                          

Поля, доступные для редактирования:

  • Системное имя
  • Формат Поля можно изменять только на аналогичный по Типу Формата Формат поля. Для того, чтобы посмотреть Тип Формата нужного вам Формата поля, клините на значок лупы Формата поля 
  • Таблица
  • Аудит включен
  • Только для чтения
  • Отобразить на форме поиска
  • Видно Пользователю
  • Показывать в списке записей
  • Перенос по словам
  • Входит в секцию
  • Обязательное поле
