Базовые команды в ОС Linux

Запуск СУБД:

systemctl start postgresql-9.6

Остановка СУБД:

systemctl stop postgresql-9.6

Перезапуск СУБД:

systemctl restart postgresql-9.6

Запуск сервера приложений:

systemctl start wildfly

Остановка сервера приложений:

systemctl stop wildfly

Перезапуск сервера приложений:

systemctl restart wildfly

Чтение логов в режиме реального времени:

tail -f /var/log/wilfly/console.log

Кар открыть порт:


systemctl start firewalld
firewall-cmd --reload

Если сервер приложений стал не доступен и не запускается, то возможно остановился сервис файерволла

Проверяется вот этой командой:

systemctl status firewalld

Для того, чтобы поднять нужно выполинть команду:

systemctl start firewalld
